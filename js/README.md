# Law AI Academy – Phase 1 Foundation

Mobile‑first premium AI learning platform.  
**Open index.html directly – no server needed.**

## Features
- 5‑tab bottom navigation (no refresh)
- Dashboard with widgets, XP, streak, level, progress
- Auto‑generated 365‑lesson list, mark completion (localStorage)
- Monthly calendar with day click popup
- Searchable notes with tags
- Tools & prompt library pages (Coming Soon)
- Dark/light mode toggle, persistent settings
- Reset progress option
- Fully responsive (iPhone SE to Desktop)
- Glass‑morphism, smooth animations

## Tech Stack
HTML5, CSS3, Vanilla JS (no frameworks)
